# grouper

A Clojure library for computing GROUPS and PERmutations

## Usage

Contains the following implemented algorithms in these namespaces

#### grouper.free-groups
- Todd-Coxeter Algorithm

#### grouper.schreier-sims
- Schreier-Sims Algorithm
